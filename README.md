## 📂 Dokumentasi Sampel (Featured Project)

### 1. User Guide: Autentikasi 2 Langkah (2FA)
* **Context:** Membuat dokumen panduan untuk fitur keamanan krusial (Autentikasi 2 Langkah) pada aplikasi SaaS fiktif.
* **Problem:** Banyak pengguna awam malas mengaktifkan 2FA karena menganggap prosesnya rumit dan takut terkunci dari akun mereka sendiri jika ponsel mereka hilang.
* **Solution:** Menyusun *User Guide* langkah demi langkah yang ramah pemula, menyertakan prasyarat aplikasi di awal, serta memberikan peringatan jelas (*blockquote*) mengenai pentingnya menyimpan *Backup Codes*.
